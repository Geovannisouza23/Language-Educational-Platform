# .NET Common Libraries

Shared libraries and utilities for .NET microservices.

## Packages

### JWT Middleware
JWT authentication middleware for ASP.NET Core

### Response Models
Standardized API response models

### Exception Handling
Global exception handling middleware

### Validation
FluentValidation extensions and helpers

## Usage

Add package reference to your .csproj:

```xml
<ItemGroup>
  <ProjectReference Include="..\..\libs\dotnet-common\DotNetCommon.csproj" />
</ItemGroup>
```
